# Bitfinex Algo T-rader

## Requirements
python3.8

## Run
(Optional) replace your Bitfinex API Key/Secret in `.env-example` and rename the file to `.env`. 
```
cd algoTrading
python3.8 -m venv ./venv
pip3.8 install -r requirements.txt
```
